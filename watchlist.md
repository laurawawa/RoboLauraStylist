# Watchlist — specific pieces RoboLaura hunts every run

Standing "in search of" (ISO) items. Each weekly run checks these in addition to
the open-ended scan, and surfaces any live match with a verified link and a
ready offer number. Apply the same verification rules as the main scan (open the
listing, confirm it's live, confirm size/color). Keep the style guide as the
backstop.

Format per item: target spec · channels · target/walk-away price · notes ·
status.
